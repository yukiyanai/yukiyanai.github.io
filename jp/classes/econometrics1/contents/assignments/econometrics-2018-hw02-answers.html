1.

粗出生率（CBR）を計算する。 ケニア：

periods <- unique(World$period)  # 期間を文字列のベクトルとして保存
Kenya_cbr <- Kenya %>%
  group_by(period) %>%
  summarize(py = sum(py.men) + sum(py.women),
            births = sum(births)) %>%
  mutate(cbr = births / py) %>%
  pull(cbr)
names(Kenya_cbr) <- periods

スウェーデン：

Sweden_cbr <- Sweden %>% 
  group_by(period) %>%

6.

スウェーデンの人口分布を用いて反事実的 (counterfactual) なケニアのCDRを算出する。 そのために、まず2005年から2010年の期間のスウェーデンの年齢別人口比率を計算する。

Sweden_pop <- Sweden %>% 
  filter(period == "2005-2010") %>%
  mutate(pop = py.men + py.women,
         ratio = pop / sum(pop)) %>%
  select(age, ratio)

求めた人口分布を使い、反事実的なケニアのCDR を計算する。

Kenya_cdr_cf <- sum(Kenya_asdr$asdr * Sweden_pop$ratio)

実際のCDRと反事実的なCDRを比べる。

c(Kenya = Kenya_cdr[2], 
  Kenya_counterfactual = Kenya_cdr_cf, 
  Sweden = Sweden_cdr[2])
##      Kenya.2005-2010 Kenya_counterfactual     Sweden.2005-2010 
##          0.010389138          0.023216456          0.009968455

この分析から、ケニアがスウェーデンと同じような人口分布だったら、ケニアの粗死亡率の方がスウェーデンのそれよりもだいぶ高くなるだろうということが推測される。

続いて、年平均何件の暗殺計画があったかを調べる。そのためにまず、国ごとの年平均暗殺計画数を数える。

ここで、年平均の数え方は複数あり得る。まず、すべての国が同じ期間観察されたと仮定しよう。 データフレームでyearの最大値と最小値を確認する。

Assassin_2 <- Leaders %>%

(dif_in_difs <- dif_killed - dif_survived)
