Python 3.7.x をWindowsにインストールする

Anaconda をインストールする

  1. Anacondaのウェブサイト を開く
  2. 画面右上の "Downloads" をクリックする
  3. "Anaconda 5.2 For Windows Installer" で、"Python 3.7 Version" の "Download" をクリックする
  4. ダウンロードの方法について尋ねられたら、「実行」を選ぶ（ダウンロードにはそれなりに時間がかかるので、気長に待つ）
  5. 通常のアプリのインストールと同じ手順でインストールを進める（チェックBOX等は既定値 [デフォルト] のままでOK。かなり時間がかかる）
    • 途中で Visual Studio Code もインストールするか尋ねられるので、必要ならインストールしておく

Jupyter Notebook を用いてPythonを学習する

Jupyter Notebook の起動

  1. インストールしたAnacondaアプリに含まれる "Anaconda Navigator" を選択して開く
  2. Jupyter Notebook（上段の左から2番目にあるはず）の "Launch" をクリックする
  3. デフォルトのブラウザが自動的に開くはず（開くアプリを尋ねられたら Edge, Internet Explorer, Google Chrome または Firefox を選ぶ）
  4. 画面の右端近くにある　”New" をクリックし、出てきたメニューから Python 3 を選ぶ
  5. Jypyter Notebook が起動するので、画面上部の Jupyter の右にある "Untitled" をダブルクリックする
  6. ノートに名前がつけられるので、わかりやすい名前をつける
  7. "File" -> "Save and Checkpoint" （またはフロッピーでディスクアイコンをクリック）でノートを保存する

Jupyter Notebookの使い方

授業で詳しく説明する。

Jupyter Notebook の終了

  1. ノートブックを保存する（"File" -> "Save and Checkpoint" の順でクリック）
  2. "File" の "Close and Halt" を選択する
  3. Jupyterのホーム画面に戻るので、右上にある "Quit" をクリックする
  4. ブラウザを閉じる

保存したNotebookを開く

  1. Anaconda Navigator を開く
  2. Jupyter Notebookの "Launch" をクリックする
  3. デフォルトのブラウザが自動的に開くので、表示されているファイルから開きたいファイル（拡張子が.pynbのもの）を選ぶ
